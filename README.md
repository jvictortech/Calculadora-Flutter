#  Calculadora em Flutter

Projeto de uma calculadora desenvolvida em **Flutter**, com foco na **separação entre interface (Front-end)** e **lógica de negócio (Back-end)**, seguindo boas práticas de organização de código.

---

##  Objetivo do Projeto

O objetivo deste projeto é:
- Construir a interface de uma calculadora utilizando widgets do Flutter
- Implementar as operações matemáticas básicas
- Separar a **tela (UI)** da **lógica de cálculo**, facilitando manutenção e entendimento do código
